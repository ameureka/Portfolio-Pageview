Linux平台环境安装脚本 参考： https://docs.facefusion.io/installation/platform/linux


### 检查现有的python环境，安装python

python的检查，如果不是Python 3.10 版本建议首先安装的python ，建议使用的python 管理工具安装，使用root 账号权限进行安装，在后续程序安装之中使用虚拟环境，保障新安装的程序与现有的程序环境不冲突。

1. ubunttu 安装pyenv 可参考之前博文， https://seealso1884.blogspot.com/2023/10/ubuntupyenv.html

2. 安装pip / git / curl / ffmpeg  等组件

可以参考：https://seealso1884.blogspot.com/2023/12/ubuntu-ffmpeg.html

```ts
apt install python3-pip

apt install git-all

apt install curl

apt install ffmpeg
```

#### 报错信息处理

来到“已安装”标签，点击“应用并重启用户界面”出现报错信息。
![Planetfall Logo](https://blogger.googleusercontent.com/img/a/AVvXsEg9MF_lbRc31mupDa9jr50fPew_FlgDsxgk-OF0B3FRD7GuhfNJrm0XRj6KDA3ANSu4swNN0n0fWn2EfT5UHwbvlu9x8n4pGzhj5CuFy5o0_CEQsVg-OCzVhh1_Cc41o4Uq_5fiR5a-eLAy1CqzSzCeZ4T-8f3ZzszNpJ-w6dLg-JBhvJsfRFhtrqnUv4k=w640-h136-rw)
看起来你遇到了一个关于 PyTorch 和 TorchAudio CUDA 版本不匹配的问题。错误信息显示 PyTorch 使用 CUDA 版本 11.8，而 TorchAudio
 使用 CUDA 版本 11.7。为了解决这个问题，你可以尝试以下几个步骤：
更新 TorchAudio： 确保你的 TorchAudio 版本与你的 PyTorch 版本兼容。在你的 diffuenv 虚拟环境中运行以下命令：

这里经过验证使用最新版本
##### CUDA 11.8版本下安装信息
```ts
pip install torch==2.1.0 torchvision==0.16.0 torchaudio==2.1.0 --index-url https://download.pytorch.org/whl/cu118
```
